# Att_ExtremeProgramming-XP-
Em duplas, criem uma aplicação simples em linguagem C, usando os conceitos de programação estudados até o momento e aplicando as práticas de
XP vistas hoje.
Escolham um escopo pequeno o suficiente para completar em uma sessão: uma lista de tarefas, uma calculadora de gorjeta, um cadastro simples ou ideia própria da dupla — sempre implementada em C.


Desafio:
Criar um programa que calcula as gorjetas de um restaurante, sendo o que deve ser levado em conta 3 coisa:
1: Valor total do pedido do Cliente
2: Quantidade de Garçons que Atenderam
3: Quantidade de Gorjeta oferecida(5%, 10%, 15%)

5% -> 0.05
10% -> 0.10
15% -> 0.15

O sistema deverá calcular em um esquema de: resultado=(ValotTotal * Gorjeta Oferecida) / QuantidadeGarçon
